"# EEG-Analysis" 

This is an EEG analysis tool that reads EDF data files of seizure from the CHB-MIT Scalp EEG Database and displays it on an interface. It currently only reads files:
chb01_01.edf
chb01_02.edf
chb01_03.edf
chb01_04.edf
chb01_05.edf
chb01_15.edf

Install the necessary files from above, libraries, and tools: mne, numpy, matplotlib, scipy, pyqt5. Run the mainf.py program.
